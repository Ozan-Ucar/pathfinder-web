# pathfinder-web

interactive pathfinding visualizer in the browser. draw walls, pick an algorithm and watch it find the shortest path.

### algorithms
- dijkstra
- bfs
- a* (manhattan heuristic)

### features
- draw walls by clicking/dragging
- drag start and end nodes around
- random maze generator
- speed control slider
- shows visited nodes and path length after search

### usage
just open `index.html` in a browser. no build step needed.
